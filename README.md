# geister

## server 端
1. 下載 server.py & room.py & common
2. 在 common 中的 constants.py 設定 SERVERHOST (server IP)
3. terminal run `python3 server.py` 

## players
1. 下載 client_gui.py & common
2. 在 common 中的 constants.py 設定 SERVERHOST (server IP)
3. terminal run `python3 client_gui.py`
```
  需要兩個玩家才能開局!!!
```
